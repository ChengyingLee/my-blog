Report 1

August 28, 2022 by Li Chengying

This is the content of my Report

Report the concrete example that changed from buying a product to using a service.

Product name

Google map

Overview

In the past, if we were looking for a place, we needed to buy maps. If we use some apps such as google map on the mobile phone, we can not only locate quickly, but also use navigation and other functions.

Merit / demerit

Its advantages are convenient, fast and low cost (only a little amount of data is needed) and it is real-time updatable. Due to the need for multi-faceted collaborative services, such as communication satellites, GPS, networking and so on, once one of them loses connection, positioning or navigation functions cannot be used.

Technologies the service uses

  • Satellite navigation：The satellite receiver calculates the user's position according to the received signal and transmits the data to the user.
  • Network communications：Through some network protocols, users and servers are allowed to interact and update data.
  • Other device in mobile phone：such as Gyroscope and Accelerometer that help measuring and calculating.
